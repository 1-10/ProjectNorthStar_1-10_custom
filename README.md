# ProjectNorthStar 1-10 custom

## はじめに
Leap Motion社とexiii社のPROJECT NORTH STARを元に、1-10社が高解像度化と業務要3Dプリンター用に設計を改良したものです。

![](/Hardware/imgs/ProjectNorthStar_1-10_custom.png)

##変更点
- ディスプレイを3.81inch AMOLED(H380BDNO1.1)に変更
- 基板固定パーツの追加
- 装飾パーツの追加
